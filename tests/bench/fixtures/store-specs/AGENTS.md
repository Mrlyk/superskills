# store-app

A small storefront pricing library that computes order totals in integer cents with typed application errors (`AppError`). Written in JavaScript ESM, tested with the built-in `node:test` runner.

## Key commands
- `pnpm install` — install dependencies
- `pnpm test` — run test suite
- `pnpm lint` — lint `src/`
